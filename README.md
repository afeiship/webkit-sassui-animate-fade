[![license][license-image]][license-url]

## installation
```shell
npm i @jswork/wsui-animate-fade
```

## usage
```scss
// use sass
@import '~@jswork/wsui-animate-fade/dist/index.scss';
// use css
@import '~@jswork/wsui-animate-fade/dist/style.css';
```
